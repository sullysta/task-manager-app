Quick Setup Guide

  1. Create the data repository
    Go to GitHub → New repository → Name: task-manager-data → Private → Create
  2. Add the initial data files
    Upload the 3 JSON files from the task-manager-data/ folder in this project (personal/tasks.json, church/tasks.json, work/cases.json)
  3. Create a Personal Access Token
    GitHub → Settings → Developer settings → Personal access tokens → Fine-grained tokens → Generate new token → Repository access: Only task-manager-data → Permissions → Contents: Read and Write
  4. Enter your settings above and tap Save
  5. Install on Android
    Open this page in Chrome → tap the Chrome menu (⋮) → "Add to Home Screen" or "Install App"

Claude Desktop MCP (Optional)

Add to %APPDATA%\Claude\claude_desktop_config.json to manage tasks by talking to Claude:

{
  "mcpServers": {
    "github": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-github"],
      "env": {
        "GITHUB_PERSONAL_ACCESS_TOKEN": "your_token_here"
      }
    }
  }
}

Then ask Claude: "Show my overdue work cases" or "Add urgent personal task: call DA office, due tomorrow"
